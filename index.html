My Projects

osu!simulation tools API

View

An API used to simulate plays on osu!standard, osu!mania, osu!catch and osu!taiko game modes and return the detailed hypothetical score. Also used to convert a total number of PP in either of these gamemodes into the approximate rank you'd get with that number of PP, and vice-versa.

Osu PP Theorizer

View

A full stack application that helps you search for a user and displays his user profile in a React page almost identical to osu!'s official webpage. It integrates a FastAPI backend that uses my osu! simulation tool API to help simulate score in different gamemodes with different parameters and returns the hypothetical profile of the user if he managed to pull off this score, and it gets displayed on the page.

Lino

View

A mobile application that connects university students with campus book boxes, allowing them to exchange, donate, or collect books. The app features an interactive map showing book box locations across campus, detailed book information pulled from Google Books API, and a user-friendly interface for browsing available titles. Users can track book availability in real-time, receive notifications when books of interest become available, and contribute to a sustainable campus community by giving their unused books a second life.
